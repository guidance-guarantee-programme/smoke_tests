# Smoke Tests

Smoke tests for [Pension Wise].


## Prerequisites

* [Bundler]
* [Git]
* [Ruby 2.2.0][Ruby]


## Installation

Clone the repository:

```sh
$ git clone https://github.com/guidance-guarantee-programme/smoke_tests.git
```

Setup the application:

```sh
$ ./bin/setup
```

## Usage

To start the application:

```sh
$ ./bin/foreman s
```
